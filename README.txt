TO VIEW THE PAGE:
-----------------------------

1| Install a Web server in local host (MAMP);

2| Connect the web server at the folder project;

3| Run from Browser Web the URL whit a parameters for get function of PHP: localhost/index.php?name=name&surname=surname&badword=badword;

4| name, surname and badword are parameters to be entered manually in the url; 

5|badword, is the parameter of a word to find and replace in php string's;
